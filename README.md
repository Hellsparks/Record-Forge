# Record Forge

Design printable vinyl record art in your browser and export a circular PNG for [HueForge](https://hueforge.com) multi-colour 3D printing.

**[Open Record Forge](https://hellsparks.github.io/Record-Forge/)**

## What it does

- Upload album art and position it inside a vinyl record template
- Add artist / album text with font, size, colour, bold, italic and arc options
- Embed a QR code or Spotify Code (paste any link)
- Choose a disc preset — LP 12″, 7″ Single, or CD
- Save projects to your browser library with thumbnail previews
- Export a circular PNG sized for HueForge (CD and 7″ exports are automatically padded so HueForge scales them to the correct physical diameter)

## No install needed

Everything runs in the browser — just open the link above or double-click `index.html` locally.
